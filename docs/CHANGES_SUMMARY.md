## üîç Solana Specifications

### Supported Network
- **Blockchain:** Solana (Mainnet)
- **Network:** Fixed to "solana"

### Supported Cryptocurrencies
- **SOL** - Native Solana token
- **USDC** - USD Coin (SPL token)
- **USDT** - Tether (SPL token)

### Wallet Address Format
- **Length:** 32-44 characters
- **Encoding:** Base58
- **Character Set:** `1-9, A-H, J-N, P-Z, a-k, m-z`
- **Excluded:** `0, O, I, l`
- **Regex:** `/^[1-9A-HJ-NP-Za-km-z]{32,44}$/`

### Example Valid Addresses
```
7xKXtg2CW87d97TXJSDpbD5jBkheTqA83TZRuJosgAsU
EPjFWdd5AufqSSqeM2qN1xzybapC8G4wEGGkZwyTDt1v
Es9vMFrzaCERmJfrF4H2FYD4KCoNkY11McCe8BenwNYB
```

## üìä Technical Architecture

### Frontend Flow
```
User Input ‚Üí Validation ‚Üí API Call ‚Üí Receive client_secret ‚Üí 
Initialize Widget ‚Üí Mount to DOM ‚Üí User Payment ‚Üí Transaction Complete
```

### Backend Flow
```
Receive Request ‚Üí Validate Parameters ‚Üí Call Stripe API ‚Üí 
Create Onramp Session ‚Üí Return client_secret
```

### API Endpoint
**POST** `/api/create-onramp-session`

**Request:**
```json
{
  "wallet_address": "7xKXtg2CW87d97TXJSDpbD5jBkheTqA83TZRuJosgAsU",
  "destination_currency": "sol",
  "destination_network": "solana"
}
```

**Response:**
```json
{
  "client_secret": "cos_xxx_secret_xxx"
}
```
